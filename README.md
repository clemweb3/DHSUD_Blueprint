## 🏗️DHSUD Blueprint 2026-2028

Building Communities, Transforming Lives

A high-fidelity, interactive dashboard proposal for the Department of Human Settlements and Urban Development. This project visualizes a new housing framework for the Philippines, focusing on integrated townships, transit-oriented development, and sustainable financing models.

🛠️ Tech Stack

- Framework: React (Vite)

- Charts: Recharts (D3-based responsive charts)

- Styling: CSS-in-JS & Standard CSS

-  Deployment: GitHub Pages
